# WIFI-Controlled-Car
WIFI Controlled Car With NodeMCU (ESP8266)

materials:
nodemcu, 4 dc gear motor with whell, L298N motor driver, lipo battery 2s 7.4V 25c,  pcb & pin female 2x15 pin, dc cable, styrofoam
